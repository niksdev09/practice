# Практика по компьютерному зрению с YOLOv8. Тема 4.

Практическая работа по изучению и применению модели YOLOv8 для задач детекции объектов. Проект демонстрирует различные возможности работы с моделью от базовой детекции до оценки качества на собственном датасете.

## Архитектура модели

В проекте используется модель **YOLOv8** (You Only Look Once version 8) - современная нейронная сеть для детекции объектов в реальном времени. В проекте представлены две версии:
- `yolov8n.pt` - nano версия (самая быстрая, но менее точная)

### Как устроена YOLOv8

Модель YOLOv8 работает в три этапа:

1. **Backbone** 
   - Берет картинку и находит в ней важные детали
   - Работает как "глаза" модели - смотрит на изображение и выделяет ключевые части

2. **Neck** 
   - Соединяет информацию с разных уровней
   - Помогает найти как большие, так и маленькие объекты на картинке

3. **Head** 
   - Делает финальные выводы - где объект и что это за объект
   - Рисует прямоугольники вокруг найденных объектов и говорит, насколько уверена в своем ответе

## Датасет

### COCO Dataset
Для этапов 1-8 используется предобученная модель на датасете COCO, который содержит 80 классов объектов.

### Собственный датасет фруктов
В 8 этапе используется датасет с фруктами:
- **Классы**: apple (яблоко), banana (банан), orange (апельсин)
- **Формат**: изображения в формате JPG с XML-аннотациями в формате PASCAL VOC
- **Расположение**: `Этапы/8/dataset/`

## Описание этапов

- **Этап 1** - Базовая детекция: простейший пример использования YOLOv8 для детекции объектов на изображении
- **Этап 2** - Извлечение данных: получение детальной информации о найденных объектах (координаты, уверенность, классы)
- **Этап 3** - Обработка видео: применение детекции к видеопотоку с сохранением результата
- **Этап 4** - Фильтрация классов: детекция только определенных типов объектов (например, только людей)
- **Этап 5** - Реальное время: детекция с веб-камеры с подсчетом количества объектов каждого типа
- **Этап 6** - Область интереса: детекция только в заданной области кадра
- **Этап 7** - Порог уверенности: демонстрация влияния порога уверенности на количество детекций
- **Этап 8** - Оценка качества: расчет метрик (Precision, Recall, F1) на собственном датасете фруктов

## Установка и запуск

### Требования
- Python 3.8+
- CUDA-совместимая видеокарта (опционально, для ускорения)
> [!IMPORTANT]
>### Установка
> 
>Установите необходимые зависимости:
>```bash
>pip install ultralytics opencv-python
>```
>
>### Запуск
>
>Каждый этап запускается независимо. Перейдите в нужную папку и запустите скрипт:
>
>```bash
>cd Этапы\1
>python main.py
>```

Для этапов с видео или веб-камерой:
- Этап 3: убедитесь, что файл `video.mp4` находится в папке
- Этап 5, 6: требуется подключенная веб-камера

### Примечания

- Результаты детекции на изображениях сохраняются с суффиксом `_result`
- Модели YOLOv8 автоматически скачиваются при первом запуске, если их нет в папке
- Для ускорения работы рекомендуется использовать GPU с поддержкой CUDA
